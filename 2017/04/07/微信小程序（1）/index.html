基础介绍

项目结构

文件夹 作用
images 用来存放图片资源
pages 用来存放页面相关文件包括：js、json、wxml、wxss
style 用来存放样式
util 用来存放工具类js

基本文件

文件类型 作用
js 页面逻辑处理、数据获取、事件响应等等
json 页面配置文件，负责窗口颜色等等
wxml 类似android的xml页面布局文件
wxss 是一套样式语言，用于描述 WXML 的组件样式。用来决定 WXML 的组件应该怎么显示。

全局文件

文件名称 作用
app.js 小程序(全局)逻辑
app.json 小程序(全局)公共设置，决定页面文件的路径、窗口表现、设置网络超 时时间、设置多 tab 等
app.wxss 小程序公共(全局)样式表
